What is the difference between map(), forEach(), filter(), find() ?

map() : map() method loops through each element in array and calls the provided function for each element.map() method returns a new array

forEach() : forEach() method loops through each element in array and calls the provided function for each element.forEach() method returns undefined

filter() : We can filter any element of the array according to the condition through javascript filter() method

find() : We can filter any element of the array according to the condition through javascript find() method.But it's returns just first filtered element.

Local Storage vs Session Storage vs Cookies

Local Storage : Local storage storing data without any expiration date.It's data is not deleted when browser is closed.It's data is available for future session.

Session Storage : Local storage storing data for a session.It's data is deleted when browser is closed.It's data is not available for future session.

Cookies : Cookies are small text files kept on a web server that identify the person using a web page.Cookies Has different expiration dates (both the server or client can set up expiration date).

What is an event loop and how it is works?

The Event Loop has one simple job to monitor the Call Stack and the Callback Queue. If the Call Stack is empty, the Event Loop will take the first event from the queue and will push it to the Call Stack, which effectively runs it. Such an iteration is called a tick in the Event Loop.
